# 🦷 Al-Noor Dental Clinic Website

Modern dental clinic website built using React + Vite + Tailwind CSS.

## 🚀 About The Project

This project is a fully responsive dental clinic website designed for academic presentation and portfolio purposes.

It includes:

- Arabic RTL Support
- Appointment Booking Form
- Admin Dashboard UI (Simulation)
- Responsive Design
- Modern Medical Theme
- Tailwind CSS Styling
- React Router Navigation

---

## 🛠 Technologies Used

- React.js
- Vite
- Tailwind CSS
- React Router
- Lucide Icons
- Framer Motion

---

## ▶️ How To Run

Install dependencies:

npm install

Run the project:

npm run dev

Then open:

http://localhost:5173
